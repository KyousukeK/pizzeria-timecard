# 🍕 ピザ屋タイムカードアプリ 仕様書

## 📌 概要

本アプリは「ピザ屋の店内で働いている体験」を再現したWebアプリであり、
壁の時計やレジ操作を通じて勤怠管理（出勤・休憩・退勤）を行う。

---

## 🎯 目的

* 楽しく直感的に勤怠管理を行う
* UI体験を重視したアプリ開発
* React / Next.js の学習

---

## 🧩 機能一覧

### ① 店内画面（メイン）

* ピザ屋の店内背景表示
* 日付表示
* アナログ時計表示（クリック可能）
* レジクリックエリア

---

### ② アナログ時計機能

#### 概要

壁に設置された時計として表示される

#### 機能

* 現在時刻をリアルタイム表示（1秒更新）
* 秒針・分針・時針あり
* 数字（1〜12）表示
* クリック可能

#### UI

* 円形デザイン
* 影付き（box-shadow）
* クリック時に縮小アニメーション

#### 動作

クリック時：

* 打刻処理実行
* 音（click.mp3）再生
* 押下アニメーション

---

### ③ 勤怠打刻機能

#### ステップ管理

以下の順番でループ：

1. 出勤
2. 休憩開始
3. 休憩終了
4. 退勤

#### 動作

* 時計クリックで順番に打刻
* localStorage に保存

#### データ構造

```
{
  id: number
  type: string
  time: string
}
```

---

### ④ レジ画面

#### 遷移

* 店内のレジクリックで移動

#### 機能

* 日付・時刻表示
* 手動打刻ボタン

  * 出勤
  * 休憩開始
  * 休憩終了
  * 退勤
* 店内へ戻るボタン

---

### ⑤ 勤怠履歴（レシートUI）

#### 概要

打刻履歴を「レシート風UI」で表示

#### UI特徴

* モノスペースフォント
* 点線ボーダー
* 白背景
* 縦スクロール

#### 表示内容

* 日付
* 時刻
* 打刻履歴一覧
* 総件数

#### 表示例

```
🧾 PIZZA SHOP RECEIPT
DATE: 2026/03/18
TIME: 12:00:00

1. 出勤
  2026/03/18 09:00

2. 休憩開始
  2026/03/18 12:00

TOTAL RECORDS: 2
```

---

#### 機能

### ⑦ タスク機能

---

### ⑧ カレンダー（仮）

## ⚠️ 注意点（重要）

### ① JSX構造

* divの閉じ忘れ禁止
* mapの外にコードを書かない

### ② コンポーネント重複

* 同じコンポーネントを複数書くと複数表示される

### ③ CSS記述

* background-image は body の中に書く
